# Compilar + Executar

Para executar o pojeto, execute na pasta raiz

``` $ make all```

# Compilar

Para só compilar o projeto, execute na pasta raiz

``` $ make release```

# Executar

Para executar, compilar o projeto e, executar na pasta raiz

``` $ make run```

# Clean

Para limpar executar na pasta raiz

``` $ make clean```
